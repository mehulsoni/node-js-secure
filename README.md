
### Node JS JWT + Server


### Server Port 8080

### Server base URL
*http://localhost:8086/*

### Installation 
Please update mongodb db url in ../db.js file
'mongodb://root:root@localhost:27017/?authSource=admin'

Run below command to install

*cd /path/to/base/dir*

*npm install*

*npm start*


### API:

- */login*
- */register*
- */verify*
- */wallets/validate/message*
- */wallets//signed-messages/:address*
